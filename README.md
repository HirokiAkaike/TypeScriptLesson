# TypeScriptLesson

TypeScriptの特徴<br>
基本的には型を定義する。<br>

アロー演算子の関数リテラルの違い<br>
　⇒　Tickイベントなどのイベントブロックでのthis の定義域がことなる。<br>
 　　・アロー演算子では、thisの定義域はアロー演算子のインスタンスがthisの定義域となる。<br>
 　　・関数リテラルでは関数リテラルのブロック内ではthisの定義域が、関数リテラルを指す。<br>
   　　イベントブロック内ではうwindows　領域がthisとなる。<br>
